Extraction commands and info:

Downloaded the gzip from https://www.cs.kent.ac.uk/people/staff/djb/pgn-extract/
Extracted into a folder

pv /Users/vyshnavigunda/Desktop/db/DATABASE4U.pgn | ./pgn-extract --nomovenumbers --nocomments --novars --notags --nonags -s > extracted_moves.pgn 2> errors.log\


cat extracted_moves.pgn | pv | tr '\n' ' ' | sed 's/ 1-0/ 1-0\n/g' | sed 's/ 0-1/ 0-1\n/g' | sed 's/ 1\/2-1\/2/ 1\/2-1\/2\n/g' | sed 's/ \+/ /g' | sed 's/^ *//g' | sed 's/ *$//g' > formatted_extracted_moves.pgn\n



pv -pipe viewer to monitor progress
--nomovenumbers: Removes move numbers from the output.
--nocomments: Removes comments from the PGN file.
--novars: Removes variations (alternative move lines) from the output.
--notags: Removes PGN header tags (like player names, dates, etc.) from the output.
--nonags: Removes NAGs (Numeric Annotation Glyphs) are numeric symbols used to annotate and evaluate chess moves. They provide information about the quality of the moves, strategy etc (example $1,$2)
-s: Runs in silent mode, so the process won’t output each game as it’s being processed (reduces terminal output).

2> errors.log:Redirects any errors that occur during the extraction process to an errors.log file instead of printing them to the terminal.


This cat command takes the PGN data from extracted_moves.pgn converts it into a single line, adds line breaks after game results (1-0, 0-1, 1/2-1/2), cleans up extra spaces.
